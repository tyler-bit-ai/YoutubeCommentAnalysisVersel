Quick Guide

프로그램 안내

Input

키워드 검색

예: 일본 로밍, 베트남 유심, SKT baro 같은 검색어로 관련 YouTube 영상을 모읍니다.

Output

카드 대시보드

조회수, 댓글 수, 구독자 대비 조회수 배율을 기준으로 영상 후보를 빠르게 비교할 수 있습니다.

Insight

전략형 분석

대표 댓글, 장단점, 실행 전략과 핵심 미션까지 한 번에 정리합니다.

분석 로직

실제 화면 동작 순서를 기준으로 요약했습니다.

1. 영상 후보 수집

YouTube 검색 API로 키워드 관련 영상을 가져오고, 영상/채널 통계를 합쳐 카드 목록을 만듭니다.

2. 정렬 점수 계산

Relevance, 최신성, 조회수를 조합한 Hybrid 점수를 계산해 우선 볼 영상을 정렬합니다.

3. 댓글과 답글 수집

선택한 영상의 상위 댓글과 답글을 함께 수집하고, 길이가 너무 짧은 텍스트는 분석 대상에서 제외합니다.

4. 샘플링과 청크 분석

댓글이 50개를 넘으면 무작위 50개를 샘플링하고, 길이에 따라 청크로 나눠 A15T 모델에 보냅니다.

5. 최종 Strategic Insight 합성

청크별 결과를 다시 종합해 요약, 장단점, 대표 댓글, 실행 전략, 핵심 미션을 JSON 기반으로 정리합니다.

이럴 때 유용합니다

로밍, eSIM, 해외 통신 관련 고객 반응을 빠르게 훑고, 실제 VOC 기반으로 마케팅 메시지나 개선 과제를 잡고 싶을 때 적합합니다.

해석 시 주의점

분석 결과는 수집된 댓글 표본 기준입니다. 표본 편향, YouTube 추천 편향, 댓글 부재 영상은 함께 고려해야 합니다.
